# Retail Sales Analytics Dashboard

## Project Overview
This project analyzes retail sales data to identify customer behavior, product performance, and sales trends using Python and data visualization techniques.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Features
- Data Cleaning
- Exploratory Data Analysis (EDA)
- KPI Metrics
- Sales Trend Analysis
- Data Visualization

## Key Insights
- Identified top-selling product categories
- Analyzed monthly sales performance
- Detected high-value customers
